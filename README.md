# Pennywise

A comprehensive money tracking and expense management application built with Flutter, helping users take control of their finances with smart categorization and analytics.

## Purpose

Pennywise simplifies personal finance management by providing an intuitive platform to track income and expenses across multiple sheets, visualize spending patterns, and make informed financial decisions.

## Features

### Core Features
- **Sheet-based Organization**: Create multiple sheets for different periods or purposes (e.g., "Nov 2025", "Ooty Trip")
- **Income & Expense Tracking**: Add, edit, and manage entries with detailed information (amount, category, date, time, notes)
- **Custom Categories**: Create personalized categories with color coding for better visual organization
- **Analytics Dashboard**: Visualize spending patterns and track personal financial trends
- **Expense Splitting**: Split bills and expenses with others (under development)
- **Secure Authentication**: Phone number-based authentication with Firebase

### Technical Features
- **Clean Architecture**: Maintainable codebase following SOLID principles
- **Cross-Platform**: Works seamlessly on iOS and Android
- **User Tiers**: Freemium and Premium subscription options (under development)
- **Modern UI**: Cupertino design system for a native iOS experience

## Technology Stack

- **Framework**: Flutter 3.8.1+
- **State Management**: GetX
- **Navigation**: go_router
- **Backend**: Firebase (Authentication, Cloud Firestore)
- **Architecture**: Clean Architecture with dependency injection

## Future Scope

- **Budget Planning**: Set monthly/weekly budgets with alerts
- **Recurring Transactions**: Automated tracking for subscriptions and regular expenses
- **Export & Reports**: Generate PDF/CSV reports for tax and accounting purposes
- **Multi-Currency Support**: Track expenses in multiple currencies with auto-conversion
- **Collaborative Sheets**: Share sheets with family members or roommates
- **AI-Powered Insights**: Smart spending suggestions and anomaly detection
- **Bank Integration**: Automatic transaction imports from bank accounts
- **Widget Support**: Home screen widgets for quick expense entry
